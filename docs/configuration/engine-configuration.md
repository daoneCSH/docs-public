# Engine Configuration

Engine 설정의 목적은 안정적인 런타임 암복호화 실행을 보장하는 것이다.

## 주요 설정 범주

- 서비스 포트
- Hub 연결 정보
- provider 관련 설정
- 캐시 또는 영속 저장 경로
- 로그와 상태 확인 설정

## 운영 해석

- Engine은 정책 원본을 직접 소유하지 않으므로 Hub 연결 정보가 중요하다.
- provider 설정은 런타임 실행 품질에 직접 영향을 준다.
- 상태 확인과 round trip 검증은 설치 후에도 지속적으로 유지해야 하는 운영 기준이다.
