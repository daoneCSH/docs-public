# Operational Tools

이 섹션은 DADP 운영 도구를 다룬다. 운영 도구는 Hub나 Engine처럼 플랫폼의 핵심 서비스 API를 제공하는 구성 요소가 아니라, 운영자가 제어면과 실행면을 설정, 검증, 동기화, 설치하는 데 사용하는 클라이언트와 보조 도구다.
